Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę złapać pomysł 💡! Proszę iterować po opuszczonym tekście i zwiększać predefiniowaną zmienną licznika za każdym razem, gdy znak jest samogłoską.

Ok, od czego powinienem zacząć?

Proszę zadeklarować zmienną count o wartości początkowej 0, którą zamierza Pan zwrócić na końcu. 

count = 0
...
return count
Co robić dalej?

Proszę zmienić podany adres text na małe litery za pomocą metody str.lower()... 

text.lower()

... i iterować po nim za pomocą pętli for. 

count = 0
for char in text.lower():
    ...
return count
Nadal potrzebuję podpowiedzi.

Może Pan sprawdzić, czy char jest samogłoską za pomocą operatora in. 

char in "aeiou"

Wyrażenie zwraca wartość logiczną, ale Python może zinterpretować ją jako liczbę całkowitą (False == 0 lub True == 1). Proszę więc zwiększyć licznik count o tę wartość wewnątrz pętli. 

count = 0
for char in text.lower():
    count += char in "aeiou"
            
return count
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Proszę tylko opakować swój kod w wyrażenie funkcyjne. Zaleca się używanie podpowiedzi typu dla danych wejściowych i wyjściowych.

Oto całe rozwiązanie:

def count_vowels(text: str) -> int:

    count = 0
    for char in text.lower():
        count += char in "aeiou"
        
    return count

Te same operacje, ale przy użyciu wyrażenia generatora. 

def count_vowels(text: str) -> int:

    return sum(char in "aeiou" for char in text.lower())
