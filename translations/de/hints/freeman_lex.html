Ich brauche noch einen Hinweis.

Sie können überprüfen, ob char ein Vokal ist mit dem in Operator. 

char in "aeiou"

Der Ausdruck gibt einen logischen Wert zurück, aber Python kann ihn als Ganzzahl interpretieren (False == 0 oder True == 1). Erhöhen Sie also den Zähler count innerhalb der Schleife um diesen Wert. 

count = 0
for char in text.lower():
    count += char in "aeiou"
            
return count
Wie sollte also meine endgültige Lösung aussehen?

Sie müssen Ihren Code nur in einen Funktionsausdruck verpacken. Die Verwendung von Typ-Hinweisen für Eingabe- und Ausgabedaten wird dringend empfohlen.

Hier ist die gesamte Lösung:

def count_vowels(text: str) -> int:

    count = 0
    for char in text.lower():
        count += char in "aeiou"
        
    return count

Dieselben Operationen, aber mit Generatorausdruck. 

def count_vowels(text: str) -> int:

    return sum(char in "aeiou" for char in text.lower())
