I still need a hint.

You may check if char is vowel with in operator. 

char in "aeiou"

The expression returns logic value, but Python may interpret it as integer (False == 0 or True == 1). So increment the counter count by this value inside loop. 

count = 0
for char in text.lower():
    count += char in "aeiou"
            
return count
So, how my final solution should look like?

You just need to wrap your code into function expression. Using type hints for input and output data are highly recommended.

Here is the whole solution:

def count_vowels(text: str) -> int:

    count = 0
    for char in text.lower():
        count += char in "aeiou"
        
    return count

Same operations, but using generator expression. 

def count_vowels(text: str) -> int:

    return sum(char in "aeiou" for char in text.lower())
